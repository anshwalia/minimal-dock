# Minimal Dock
#### Dock Widget to Keep Your Desktop Clean

### Tech Stack
* ElectronJS
* NodeJS
* Bootstrap 4
* Library
  * anime.js
  
### Getting Started
#### 1. Clone the repository.
#### 2. Install NodeJS.
#### 3. Open Terminal in Project Directory.
#### 4. Install required modules.
```
npm install
```
#### 5. Execution
```
npm start
```
